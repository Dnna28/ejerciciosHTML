Funcion

Una función es un bloque de código que se puede invocar y ejecutar en cualquier parte de un programa. Puede recibir parámetros de entrada y devolver un valor de salida. En la programación, las funciones se utilizan para modularizar el código, reducir la repetición y mejorar la legibilidad.

let

"let" es una palabra clave que se utiliza en JavaScript para declarar una variable local dentro de un bloque de código. Las variables declaradas con "let" tienen un alcance limitado dentro de su bloque de código y no se pueden acceder desde fuera de él. A diferencia de "var", las variables declaradas con "let" no pueden ser redeclaradas dentro del mismo ámbito.

const

"const" es una palabra clave que se utiliza en JavaScript para declarar una variable cuyo valor no cambiará durante la ejecución del programa. Las variables declaradas con "const" también tienen un alcance limitado y no pueden ser reasignadas a otro valor una vez que se les ha asignado uno.

figcaption

figcaption: Esta etiqueta se utiliza para agregar una descripción o leyenda a una imagen en una página web.

datalist

datalist: Esta etiqueta se utiliza para crear una lista desplegable de opciones que se pueden seleccionar en un campo de entrada de formulario.
